# Student Management System

A console-based Student Management System developed using Python and Object-Oriented Programming (OOP) concepts.

## Features

### Student Management

* Add Student
* Display Students
* Search Student by Roll Number
* Update Student Details
* Remove Student

### Mark Management

* Add Marks for Students
* Display Student Marks
* Calculate Total Marks

## Technologies Used

* Python
* Object-Oriented Programming (OOP)
* Dictionaries
* Modular Programming
* User Input Handling

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open the project folder

```bash
cd student_management
```

3. Run the application

```bash
python main.py
```
